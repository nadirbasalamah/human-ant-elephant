# human-ant-elephant

A simple game based on the "human, ant and elephant" suit game.

## How to Play

1. Clone this repository.

2. Play the game.

```sh
./game
```

If you are using Windows. use this command.

```sh
./game.exe
```

## How to Explore

1. Clone this repository.

2. Run the test.

```sh
go test
```

3. Play the game.

```sh
./game
```
